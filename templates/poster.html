0-Run

After having set up the basics with the files and linking it to github, we needed to have the server running to display our web-server.

To do this we ran the terminal command python3 app.py.

Initially this ran without any issues however, this soon changed. Eventually it started comming up with an error saying that the port was taken up. Now! There is an easy solve. Heading over to the app.py and scrolling to the bottom showed a little line of code which was for port allocation. By default it was working with a function that would find a port for you. Despite its valiant attempt, it did not succeed. So, I manually changed the port to a random number and for some reason I thought 55551 was a good choice! Huzzah! It worked!
